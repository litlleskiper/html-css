Texto marcado

Nesta frase temos um termo marcado usando a TAG mark

Texto grande e texto pequeno

Nesta frasa estamos criando um texto grande e um texto pequeno

Texto deletado

Podemos marcar um texto como excluido usando a TAG del, indicando que o texto foi removido, deve ser lido, mas desconsiderado.

Texto Inserido

Podemos marcar um texto como inserido usando a TAG ins, indicando que o texto foi adicionado posteriormente.

Texto sobrescrito

Podemos criar um texto sobrescrito usando a TAG sup, como em fórmulas químicas ou matemáticas.
Ex: x=232

Texto subescrito

Podemos criar um texto subescrito usando a TAG sub, como em fórmulas químicas ou matemáticas.
Ex: H2O
